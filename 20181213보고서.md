2. 금주 실습할 내용
```
  - 실습명 : nodejs 웹서버 구축
  - 실습절차 :
    가. nodejs설치
    나. nodejs를 이용하여 웹서버 구축
    다. arduino에서 웹서버 client 작성
    라. NodeMCU로 얻은 온도값을 웹서버로 전달
    마. 받은 온도값 데이터를 웹페이지에 출력
```
